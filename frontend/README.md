
  # Главная страница приложения

  This is a code bundle for Главная страница приложения. The original project is available at https://www.figma.com/design/2tjyRQurZ3o0SGBrmHbVxt/%D0%93%D0%BB%D0%B0%D0%B2%D0%BD%D0%B0%D1%8F-%D1%81%D1%82%D1%80%D0%B0%D0%BD%D0%B8%D1%86%D0%B0-%D0%BF%D1%80%D0%B8%D0%BB%D0%BE%D0%B6%D0%B5%D0%BD%D0%B8%D1%8F.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Добавление на экран телефона

  iPhone:
  1. Открыть сайт в Safari
  2. Нажать кнопку «Поделиться»
  3. Выбрать «На экран “Домой”»

  Android:
  1. Открыть сайт в Chrome
  2. Нажать три точки (меню)
  3. Выбрать «Добавить на главный экран»
